Human Genome Annotations

  • GenoCanyon (Server) (Web Application) A statistical framework to predict functional non-coding regions in the human genome through integrated analysis of genomic conservation measures and biochemical annotations.
  • GenoSkyline (Website) Tissue-specific functional annotation through integrative analysis of Roadmap epigenomic data
  • GenoSkyline Plus(Website) GenoSkyline-Plus is a comprehensive update of GenoSkyline that incorporates more annotation data into the framework and extends to 127 integrated annotation tracks covering a spectrum of human tissue and cell types.

Analysis of Genome Wide Association Study Data

  • GWAS.PC (User Manual Code) GWAS.PC (GWAS Power Calculation) is an R package that does power analysis in genome wide association studies. In particular, genotyping error is considered in power calculation.
  • GenoWAP (Website) Post-GWAS prioritization through integrated analysis of GWAS summary statistics and GenoCanyon genomic functional annotation
  • GPA (Website) A statistical approach to prioritizing GWAS results by integrating pleiotropy information and annotation data.
  • GBR (Download Example) This is an R package implementing a post-GWAS prioritization algorithm, which incorporates the rewiring information of co-expression network to prioritize GWAS signals.
  • GWAS with MRF pathway  (Related Paper) (Example.R fun_network.R network.csv pval.txt) This is a program to implement the Markov Random Field (MRF) method to incorporate pathway topology for genome wide association studies.

eQTL

  • LORS (Download) A Low-Rank representation and Sparse regression for eQTL mapping. This algorithm accounts for confounding factors such as unobserved covariates, experimental artifacts, and unknown environmental perturbations.

Pathway Analysis

  • Pathway Analysis using Random Forests (Source code and supplemental data) An R code for pathway-based classification and regression using Random Forests.
  • GRAPE (Source code and sample datasets) GRAPE is a template method that allows for identification of perturbed pathways in individual tumor samples relative to a reference collection of samples (e.g., matched healthy tissue). GRAPE is sensitive to biological variability, robust to batch effects and can be applied to any gene expression platform.
  • COSINE  (Related Paper) (Software)
    An R package to extract the globally most discriminative sub-network from multiple gene expression data sets with integration of protein-protein interactions data.

Cancer Mutation Cluster Identifications

  • iPAC (More information) iPAC (Identification of Protein Amino acid mutation Clustering) finds mutation clusters on the amino acid level while taking into account the protein structure.
  • GraphPAC (More information) A bioconductor R package for identifying mutational clusters of amino acids in a protein while utilizing the protein tertiary structure via a graph theoretical model.

Pathway Tools

  • VitaPad  (Link to the software) A program to visualize and analyze pathways.

Drug Target Identifications

  • iFAD (Software) (DataSet)
    An R package implementing a bayesian sparse factor model for the joint analysis of paired datasets, the gene expression and drug sensitivity profiles, measured across the same panel of samples, e.g. cell lines.
  • FacPad (Software) (More Information)
     An R package implementing a Bayesian sparse factor model for the inference of pathways responsive to drug treatment.

Haplotype Analysis

  • Haplore (Paper) (ReadMe - PDF)  (Download-Windows Download-Solaris Download-Linux) A Program for Haplotype Reconstruction in Pedigrees
  • Overall LD (Paper) (Download) A program to assess the overall Linkage Disequilibrium (LD) over several markers and assess the LD between groups of markers.

TDT

  • Multiple TDT (Paper) (Download) A program to do the associate test (TDT type test for nuclear-family data) using several tightly linked markers simultaneously
  • QSAT (Paper) (ReadMe - PDF) (Download) A Program for Quantitative Association Analysis

Linkage Disequilibrium Mapping

  • LD Mapping with Haplotype Data (Paper) (Download) A program of mapping disease genes (including location and confidence interval) by using haplotypes with disease mutation and the haplotypes without disease mutation (or allele frequencies in normal population).
  • LD Mapping with Genotype Data (Paper) (Download) This is a program to mapping the disease gene (including location and confidence interval) by using multisite genotype data of case-control studies.

Data Integration

  • MDIntegration (More information) An R package for multi data sources integration using composite association network, relevance vector machine (RVM), or RVM-boost.

Population Structure

  • Population Structure Inference (PSMix) (Readme) (Download) An R package to infer population structure using genomic markers.

Cluster analysis

  • BCLUST (Paper) (ReadMe - PDF) (Download) A program to assess reliability of gene clusters from expression data by using consensus tree and bootstrap resampling method

Sequencing Data Analysis

  • RNASeqBias (Download Manual Vignette) An R package for detecting and correcting biases in RNA-Sequencing data. The download link contains the package, the manual and a "vignette" for quick start.
  • pairedBayes (Download ReadMe.txt Example Rdat file) An R code for Bayesian modeling of paired RNA-seq experiment.
  • Psafe (Download) A pipeline to adjust the biases and errors of sequencing data to gain more accurate allele frequency estimation.

Genotype Calling from Microarrays

  • M3 and M3-S (Download) A combined package for M3 and M3-S - an improved SNP calling algorithm for Illumina BeadArray Data.

MicroRNA-gene interactions

  • MCMG (Download) MCMG is designed to infer the gene-microRNA interaction pairs from joint analysis of multiple cancers.

Transcription Factor Binding using Phylogeny

  • TFBphylo (Download) An R package implementing a phylogenetic tree based method to model the on/off rates of transcription factor binding (TFB) events, and uses the EM algorithm to estimate model parameters.

Proteomics Data Analysis

  • MRM pre-processing (Download) Workflow for data pre-processing of multiple reaction monitoring (MRM) experiments. It includes data quality assessment and normalization.
  • Graphics for MRM and SWATH (Download MRM & SWATH Graphics Manual iTRAQ code iTRAQ Manual) An R package providing exploratory graphics and qualify assessment for MRM data generated from Multiquant (ABSciex) and SWATH datasets.
  • MSq (Download) Data Pre-processing for label-free multiple reaction monitoring (MRM) experiments (Quality assessment and data normalization)

Microbiome Analysis

  • MSClust (Download) An M package for Clustering 16S rRNA sequences into OTUs. The download link contain the package and some benchmark data sets.

Human Nondisjunciton Data

  • EM based method (Paper) (Download) An EM based algorithm to estimate genetic distances and chiasma distribution for human UPD and trisomy data.
  • HMM based method (Paper) (Download) A Hidden Markov model based algorithm to estimate genetic distances and chiasma distribution for human UPD and trisomy data.

Modifier Gene Analysis

  • siliColony (Download) A program to estimate the probability that a modifier gene is responsible for the segregation of phenotypes.

Others
